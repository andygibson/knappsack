To install the archetypes, in a command line, run :

mvn clean install

This will install the maven archetypes into your local repository where you can use them to create new maven projects.

For more information, please read the getting started guide at

http://www.andygibson.net/blog/tutorial/getting-started-with-knappsack-archetypes/

You can read more about the Knappsack archetypes at :

http://www.andygibson.net/blog/tag/knappsack/
